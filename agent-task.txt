Sen bir senior full-stack yazilim muhendisisin. SwarmUI projesini gelistireceksin.
Proje: Portainer Business Edition alternatifi (NestJS + React + MongoDB + Docker Swarm)

Detayli plan PLAN.md dosyasinda mevcut. Onu oku ve takip et.

GOREV - PHASE 0 + PHASE 1 tam implementasyon:

1. PHASE 0: Monorepo Kurulumu
- pnpm workspace + turborepo kurulumu
- Backend: NestJS projesi (apps/backend)
- Frontend: React + Vite projesi (apps/frontend)
- packages/shared-types
- infrastructure/docker/docker-compose.yml (MongoDB + Redis)
- .env.example dosyasi

2. PHASE 1: Auth + Core Moduller
Backend moduller (tam implementasyon):
- Auth modulu: JWT access+refresh token, local login, bcrypt
- Users modulu: CRUD, password management
- Teams modulu: CRUD, uyelik yonetimi
- Roles modulu: RBAC, builtin roller (Admin/Operator/Helpdesk/Standard/ReadOnly)
- Endpoints modulu: Docker endpoint CRUD, baglanti testi
- Docker service: dockerode wrapper, multi-endpoint connection pool
- Settings modulu: temel sistem ayarlari
- Activity log interceptor (otomatik API loglama)
- Auth log servisi
- Global exception filter
- Swagger/OpenAPI dokumantasyon

Frontend sayfalar (tam implementasyon):
- Login sayfasi (JWT auth)
- App shell (sidebar navigation, topbar)
- Dashboard (placeholder metrikler)
- Users yonetim sayfasi
- Teams yonetim sayfasi
- Roles yonetim sayfasi
- Endpoints yonetim sayfasi
- Settings sayfasi
- React Query + Zustand kurulumu
- Axios interceptor (token refresh)

TEKNIK KARARLAR:
- Backend port: 3001
- Frontend port: 3000
- MongoDB: docker-compose ile localhost:27017
- Redis: docker-compose ile localhost:6379
- JWT: access token 15dk, refresh token 7 gun (HttpOnly cookie)
- Sifrelem: bcrypt (salt 12)
- UI: shadcn/ui + tailwindcss
- State: TanStack Query + Zustand
- HTTP: axios

ONEMLI KURALLAR:
- Her modulu tam ve calisir halde yaz, placeholder birakma
- TypeScript strict mode
- Her servis icin interface tanimla
- Error handling eksiksiz olmali
- PLAN.md'deki MongoDB sema tasarimini kullan
- Swagger decorator'lari ekle
